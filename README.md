# 💐 what-flower<br/>
App that can recognize what kind of flower is on the picture you took.

**This app uses:**<br/>
-MVC<br/>
-CoreML<br/>
-Cocoapods<br/>
-Alamofire<br/>
-REST API<br/>
-SwiftyJSON<br/>
